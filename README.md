## Getting Started

### Prerequisites
- Java Development Kit (JDK 8 or higher, preferably JDK 17+)
- Apache Maven (for Maven-based projects)
- An IDE such as IntelliJ IDEA or Eclipse
- Oracle SQL Developer or equivalent PL/SQL client (for PL/SQL scripts)

### Running Java Projects
For any of the subdirectories containing a `pom.xml`, you can compile and test the project using:
```bash
mvn clean test
```

### Running PL/SQL Scripts
The scripts inside `Week 1/PLSQL_Exercises/` can be executed in an SQL worksheet connected to an Oracle database instance with appropriate tables (`Customers`, `Loans`, `loanDetails`).
